# リズム戦闘PoC rAFウォッチドッグ撤回設計

作成日: 2026-06-20

## 判断

v20/v21の実機比較では、ウォッチドッグが発動していないLINE鳴動環境でもタップ後の操作感が悪化し、Android Chrome音量0の入力問題も安定して解消しなかった。常時25msタイマーを維持する利益が確認できないため撤回する。

## 変更

- 25ms視覚ウォッチドッグ、フォールバック判定、発動回数を削除する。
- 視覚描画は元の`requestAnimationFrame`経路だけに戻す。
- v21のHIT・MISS・撃破時ノーツ削除は維持する。
- 音声、判定、譜面、診断のrAF・render・timer計測は維持する。

## 検証

- ソース契約テストで視覚用25msタイマーが存在しないことを確認する。
- JavaScriptキャッシュを`v=22`へ更新し、配布フォルダを同期する。
